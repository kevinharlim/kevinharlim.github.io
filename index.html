Impression about UI/UX Design

The First

Jadi, mengapa saya lebih menyukai UI/UX Design dari pada Programming? Well, singkat cerita pada bulan Februari 2020, saya sedang membuka instagram di Smartphone, lalu di halaman home di instagram saya, muncul UI (User Interface) design Mobile dan itu bagus banget menurut saya, jadi saya cari tahu lebih dalam tentang design-design UI tersebut. Tidak hanya design mobile juga, saya juga melihat design UI Website yang meningkatkan rasa ketertarikan saya kepada UI Design. Saya mencari referensi-referensi tersebut di google dengan keyword “UI website and mobile Design”, hingga akhirnya menemukan sebuah website berisi tentang UI website dan mobile design, yaitu : Dribbble

Pada website ini, saya mendapatkan banyak inspirasi mengenai UI Design. Lalu saya belajar UI harus di dasari oleh UX(User Experience), UX itu pengalaman dari pengguna sebelumnya menggunakan sebuah website atau aplikasi. Jadi, ketika mendesign sesuatu saya juga harus mengimplementasikan UX-nya juga, agar design saya ketika ingin di build menjadi aplikasi yang user-friendly dan tidak kebingunan ketika User memakainya.

Kedua foto ini, adalah design pertama saya ketika mencoba mendesign Login page dan Homepage pada suatu Website. keduanya masih jauh dari kata sempurna, mulai dari konsistensi icon, white spacing, typography, layout. dan masih banyak lagi. tetapi saya bersyukur mencoba.
Ada pepatah mengatakan “Jangan takut gagal sebelum mencoba, jangan takut jatuh sebelum melangkah, Kesuksesan milik orang yang berani mencoba”
